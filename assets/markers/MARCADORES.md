# Marcadores para Realidade Aumentada

Este projeto usa 5 marcadores diferentes do AR.js. Imprima ou exiba estes marcadores na tela para visualizar os veículos em 3D.

## Marcadores Disponíveis

### 1. Marcador Hiro - Police Car
- **Arquivo:** `hiro.png` (já está na pasta)
- **Veículo:** Police Car (Carro de Polícia)
- **Escala:** 0.5

### 2. Marcador Kanji - Audi e-tron
- **Link para download:** [Kanji Marker](https://raw.githubusercontent.com/AR-js-org/AR.js/master/data/images/kanji.png)
- **Veículo:** Audi PB18 e-tron (Low Poly)
- **Escala:** 2.0

### 3. Marcador Letter A - Sedan
- **Link para download:** [Letter A Marker](https://raw.githubusercontent.com/AR-js-org/AR.js/master/data/images/letter-A.png)
- **Veículo:** Generic Sedan
- **Escala:** 0.3

### 4. Marcador Letter B - Sport Coupé
- **Link para download:** [Letter B Marker](https://raw.githubusercontent.com/AR-js-org/AR.js/master/data/images/letter-B.png)
- **Veículo:** Generic Sport Coupé
- **Escala:** 0.3

### 5. Marcador Letter F - Red Car
- **Link para download:** [Letter F Marker](https://raw.githubusercontent.com/AR-js-org/AR.js/master/data/images/letter-F.png)
- **Veículo:** Red Car
- **Escala:** 0.5

## Como Usar

1. Baixe as imagens dos marcadores pelos links acima
2. Imprima em papel branco (tamanho mínimo: 5cm x 5cm)
3. Ou exiba na tela de outro dispositivo
4. Abra `index-multi.html` no navegador
5. Aponte a câmera para o marcador

## Dicas

- Mantenha boa iluminação
- Evite reflexos no papel
- Mantenha o marcador plano e visível
- Distância ideal: 20-50cm da câmera
